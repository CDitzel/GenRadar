Pytorch Implementation of the paper [[https://arxiv.org/pdf/2107.08948.pdf][GenRadar: Self-supervised Probabilistic
Camera Synthesis based on Radar Frequencies]]. The model was designed to
probabilistically generate intuitive camera views of real-world scenarios if
supplied with nothing but synchronous range-Doppler information.
