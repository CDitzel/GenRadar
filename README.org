Pytorch Implementation of the paper [[arxiv_link][GenRadar: Self-supervised Probabilistic
Camera Synthesis based on Radar Frequencies]]. The model was designed to
probabilistically generate intuitive camera views if supplied with nothing but
synchronous range-Doppler information.
